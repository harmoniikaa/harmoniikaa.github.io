BZOJ 4568

直接链剖+线性基,这样的复杂度是$O(Q log^2 n \times 60^2)$,看起来很大,跑得也可能非常慢.
可以加入一个常数优化:合并线性基时采用启发式合并,这样的复杂度看起来会变成$O(Q log^2 n log 60 \times 60)$,会跑得挺快．
似乎使用点分治会获得更好的复杂度.
